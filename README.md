# Scripts

This is a repository of scripts that I use in my own personal life. I felt like they might have uses to other people as
well, so you will find them here.

## Table of Contents
1. [Convert a Youtube Playlist to a Spotify Playlist](./Youtube-playlist-to-Spotify/README.md) - Transfer your YouTube playlists to Spotify
2. [Add bleed to a Trading Card for MakePlayingCards.com](./Adding-Bleed-To-Card/README.md) - Prepare card designs for professional printing
3. [Aggregate Music Metadata from YouTube Playlist](./Aggregate-music-metadata-from-youtube-playlist/README.md) - Extract and enhance music metadata without downloading files

## Overview

These scripts cover various automation tasks including:
- **Music Management**: YouTube playlist extraction and Spotify integration
- **Design Tools**: Print preparation for trading cards
- **Data Processing**: Metadata enhancement using music databases
